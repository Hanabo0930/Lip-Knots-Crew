# Lip Knots Crew v5.6を見る

Windowsでは、ZIPを解凍して`アプリを見る_WINDOWS.cmd`をダブルクリックしてください。

- 選択画面: http://127.0.0.1:4172
- スタッフアプリ: http://127.0.0.1:4173
- 管理画面: http://127.0.0.1:4174

配布済み画面を使うため`npm install`は不要です。Node.js 22だけ必要です。終了は黒い画面で`Ctrl+C`です。デモは外部送信・本番変更を行いません。

管理画面の「未達原因・改善再実行統制盤」で、完了した経営判断の成果と、未達原因・改善案・再実行タスクを確認できます。

「EXECUTIVE OUTCOME & ROI」に成果測定率、100点評価、投資額、効果額、純効果、総合ROIが表示されます。「成果PDF」「成果CSV」もデモで動作します。PDFは印刷画面で「PDFとして保存」を選びます。

「OUTCOME RECOVERY LOOP」では、効果悪化・目標未達・ROI赤字を自動診断し、原因を番号で選ぶと改善案・期限・担当者付きの再実行タスクを作成できます。
