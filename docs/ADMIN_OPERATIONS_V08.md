# 管理実務 v0.8

## 追加した管理者機能

- スプシ書込エラー・競合一覧
- 一時的なシステムエラーの手動再試行
- 手入力競合は自動上書きせず、確認済み処理
- アプリ応募の管理者確認
- 報告書確認時の経費入力
- 一時保存
- 確認完了後の安全書込キュー
- スプシ該当行を直接開くリンク

## 経費入力列

現行シフト表:

- AK: 交通費
- AL: 8％買取
- AM: 10％買取
- AO: ネットプリント
- AP: 切手・速達・レターパック

## 上書き防止

経費確認を開いた時点の値を `expected` として保存します。
確認完了までに社長がスプシを直接変更した場合、値が一致しないため書込を停止します。

競合が起きた場合:

1. 管理画面に赤く表示
2. 現在値と書込予定値を確認
3. スプシを正として手動対応、または再度経費画面を読み直す
4. 競合を自動で無視して上書きしない
