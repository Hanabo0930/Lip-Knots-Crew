# Google Driveでの開発ファイル保存

Driveへの保存で問題ありません。むしろ、このチャットだけに残さず会社のDriveへ保管するのが安全です。

## 推奨フォルダ

Lip Knots Crew
- 01_現行最新版
- 02_旧バージョン
- 03_仕様書・UI
- 04_営業資料・料金
- 05_監査・移行
- 06_素材・ロゴ
- 90_本番設定メモ
- 99_廃止候補

## 運用

- 新版ZIPは `01_現行最新版` へ保存
- 一つ前のZIPを `02_旧バージョン` へ移動
- v0.8はv0.1～v0.7を含む累積版なので、実装時は最新版を使う
- 旧版は戻すために残す
- ZIPは解凍せず原本も残す
- ファイル名のバージョン番号を変えない

## 保存してはいけないもの

次の秘密情報は、通常の共有フォルダやZIPへ入れません。

- Firebaseサービスアカウント秘密鍵
- Gmail送信用サービスアカウントJSON
- APIキーの秘密値
- VAPID秘密鍵
- 本番環境のSecret
- 個人情報入りの検証データ

秘密情報はFirebase Secret Manager、Google Cloud Secret Manager、またはアクセス制限した専用保管場所で管理します。
