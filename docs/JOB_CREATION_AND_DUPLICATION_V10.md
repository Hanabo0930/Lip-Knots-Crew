# 案件追加・複製 v1.0

## 案件追加

管理画面から次を入力できます。

- 実施日
- クライアント
- 店舗
- メーカー
- メニュー
- 入店時間
- 実施時間
- 外注名
- 募集人数 1～20名
- 基本単価
- 公開方法

募集人数が3名なら、同じ内容の案件を3枠作成します。
各枠は別のjobIdを持ち、同時応募でも1枠につき1名だけ確定します。

## 安全状態

`companyFeatureSettings/{companyId}.adminJobCreationSourceReady=false` の場合、
管理画面で作った案件はFirestoreへ下書き保存されます。

スプシへ新規行を作る工程が完成するまで、即時公開を指定しても自動的に下書きになります。
これにより、アプリとシフト表の件数が勝手にずれることを防ぎます。

## 複製

既存案件から基本情報と入力金額を複製できます。

複製しないもの:

- スタッフ手配
- 事前連絡
- 提出物
- キャンセル
- 応募履歴
- 端末・通知履歴
