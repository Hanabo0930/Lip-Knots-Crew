# ログイン案内・端末管理の導入順

1. Firebase Authenticationでメールリンク認証を有効化
2. `staff.lipknots.com` を承認済みドメインへ追加
3. Gmail API・Workspace委任を設定
4. SecretとURLパラメータを設定
5. Functionsをデプロイ
6. 管理者1名・テストスタッフ2名でログインメールを確認
7. 1時間後にリンクが失効することを確認
8. iPhone・iPad・PCで複数端末ログイン
9. 個別端末ログアウト
10. 管理者の全端末ログアウト
11. 未ログイン対象抽出
12. 5～10名へ案内メール
13. 問題なければ30～50名
14. 最後に全スタッフへ展開
