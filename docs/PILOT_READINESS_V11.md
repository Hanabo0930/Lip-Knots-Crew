# 本番導入チェック v1.1

管理画面に「本番導入チェック」を追加しています。

## チェック項目

1. シフト表の読取同期
2. スタッフ名簿の読取同期
3. 安全書込マッピング
4. 新規行追加
5. 案件ID列
6. 管理画面からの案件追加
7. 停止中・手動確認キューがない

## 行追加の事前検査

スプシを変更せず、次を確認します。

- 対象月タブがある
- 最後の有効案件行を判定できる
- 雛形行に必要な数式がある
- 必須入力規則がある
- 追加予定行を計算できる
- 条件付き書式の雛形がある

## 表示

- 導入可能: 全項目合格
- 検証中: OFF設定または未確認項目あり
- 要確認: 数式、入力規則、月タブなどの不一致
