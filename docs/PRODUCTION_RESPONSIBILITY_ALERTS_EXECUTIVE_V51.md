# 本番責任運用アラート・週次経営レポート v5.1

## 自動基準

| 指標 | 基準 |
| --- | ---: |
| 責任承認率 | 90%以上 |
| 期限内承認率 | 80%以上 |
| 承認時間P95 | 120分以内 |
| 再割当成功率 | 70%以上 |
| 管理者緊急移管 | 0件 |
| 担当負荷差 | 2件以内 |
| 承認待ち比率 | 20%以内 |

基準差からWARNING・CRITICAL、リスク点、即時対応を決定的に生成します。判定指紋が同じなら入力データと判断内容も同じです。

## 週次経営レポート

直近8週、前週差、主要KPI、成果、リスク、経営判断をMarkdownへまとめます。管理画面のコピーまたはMD出力を使用します。

## CSV

CSVには集計、週、担当、担当週、アラートを含みます。UTF-8 BOM、引用符エスケープ、式注入防止を適用し、生UID・Push token・秘密値は含めません。

## 本番確認

1. 実Firebase productionへv5.1を承認付きデプロイする。
2. 8週分の`weeklyReports/recipients`を同期する。
3. アラートの現在値・基準値と元証跡を照合する。
4. MDとCSVを出力し、経営判断・日本語・式注入防止を確認する。
5. 受入失敗時は既知正常版v5.0.0へ別承認rollbackする。
