# 本番SLA前週比較・週次配信・期限督促 v4.5

## 自動運用

- `monitorProductionReleaseEvidence`: 1分ごとにSLAとアラート状態を監視
- `runProductionEvidenceOperationsDigest`: 60分ごとに週次配信とレビュー督促を評価
- 週次レポート: 毎週月曜09:00 JST以降、企業・証跡・週につき1回
- レビュー督促: 期限24時間前または期限超過時、レビュー・日付につき1回
- 期限超過: 緊急扱いで静穏時間をバイパス

## 前週比較

今週はJST当日を含む直近7日、前週はその直前7日です。期間は重複しません。総合点、確認SLA、対応開始SLA、総件数、超過、平均解決時間、レビュー未完了を比較します。

## 証跡

`weekly_report_delivered`と`after_action_review_reminded`を証跡イベントへ保存し、同時に`auditLogs`へsystem実行を記録します。クライアントの直接書込は常に拒否します。
