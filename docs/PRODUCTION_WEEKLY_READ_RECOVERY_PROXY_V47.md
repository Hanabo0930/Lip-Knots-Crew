# 本番週次レポート 自動復旧・代理確認・既読SLA推移 v4.7

## 自動復旧

- 60分ごとの運用処理で直近8週の`unresolved`配信先を再確認
- `ADMIN_EMAILS`のFirebase Auth UIDが見つかった場合だけ`unread`へ復旧
- 未解決件数を減算し、未読件数を加算して本人へレポートを再配信
- 復旧イベント、監査ログ、解決時刻、通知queue IDを保存

## 代理確認

- 通常既読はログイン本人の自分宛てだけ
- 代理確認は対象が`unresolved`または最大LEVEL 3の場合だけ
- 実行者は本番管理者allowlistに限定し、自己代理は禁止
- 10〜500文字の理由、代理確認者、対象メール、実行時刻を監査保存
- 本人既読・代理確認済みはどちらも既読率へ計上し、代理件数は別集計

## 既読SLA推移

- 直近8週を週キー順に並べ、既読率、代理確認、未読、未解決、LEVEL 3を集計
- 現在値、前週値、差分、平均、改善・維持・低下を決定的に算出
- UIDは管理画面へ返さず、Firestoreクライアント直接書込は拒否
