# 週次既読SLA予測・責任者自動割当 v4.8

## 自動判定

- 既読SLAは配信から24時間
- 配信後12時間未満は安全、12時間以上は要監視、20時間以上は期限超過予測
- 配信先未解決は時刻に関係なく期限超過予測
- 24時間以降はSLA超過

## 自動割当

- 未解決は即時、通常未読は配信20時間後に実行
- 対象本人を除く`ADMIN_EMAILS`から企業・週・対象者に対して決定的に責任者を選定
- 同じ対象への再実行は既存割当を保持
- 責任者へPush通知し、割当理由・リスク点・SLA期限を監査保存

## 責任者承認

- 自動割当された責任者本人だけが承認可能
- 10～500文字の引受理由が必須
- 確認済み対象、第三者、別企業、クライアント直接書込を拒否
- 承認者・承認時刻・理由をイベントと監査ログへ保存

## 確認

管理画面の`READ SLA RESPONSIBILITY`で、リスク点、期限、担当者、承認状態を確認します。本人既読または許可条件を満たす代理確認で完了します。
