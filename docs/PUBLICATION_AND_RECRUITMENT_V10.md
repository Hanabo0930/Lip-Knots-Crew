# 募集公開・停止・公開予約 v1.0

状態:

- draft: 下書き
- scheduled: 公開予約
- open: 募集中
- stopped: 募集停止
- assigned: 手配済み
- cancelled: キャンセル

## 公開条件

- キャンセルではない
- 未手配
- 必須情報がある
- スプシ由来の案件、または新規案件の安全なソース作成が有効
- 募集停止ではない

## 公開予約

5分ごとのスケジューラーで予約日時を過ぎた案件を公開します。
予約時刻になっても安全条件を満たさない場合は公開せず、理由を保存します。

## 募集停止

募集停止は案件を削除しません。
スタッフの募集一覧から隠し、管理画面と集計には残します。
