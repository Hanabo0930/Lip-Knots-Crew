# 管理画面の安全編集 v1.0

## 編集可能

- スタッフ
- クライアント
- 店舗
- 店舗住所
- 最寄駅
- メーカー
- メニュー
- 入店時間
- 実施時間
- 外注名
- 請求側 S～Zの入力セル
- 支払側 AB～AIの入力セル

## 編集禁止

- AA 売上合計
- AJ 基本給合計
- AR 支給額合計
- BB 外注合計
- 数式セル
- GAS生成セル
- 行全体

## 競合防止

案件にrevision番号を持ちます。
別画面で先に保存された場合、古い画面からの保存を拒否して再読込を求めます。

スタッフ変更時:

- 現役スタッフだけ選択可能
- 同日重複を確認
- 旧スタッフの同日ロックを解除
- 新スタッフの同日ロックを作成

## スプシ書込

安全書込設定がOFFなら、アプリ側へ保存して `pendingSourceWrite` を付けます。
設定がONで該当行を確認できる場合だけ、許可列をキューへ送ります。

店舗住所と最寄駅はFirestoreのアプリ管理項目です。既存スプレッドシートに対応列を追加せず、スタッフ画面の地図・公共交通経路に利用します。所在地だけの更新はスプシ書込キューに送りません。
