# 月別タブへの安全な新規行追加 v1.1

## 対象

管理画面で新しく作成・複製した案件を、実施日の月別タブへ追加します。

例:

- 実施日 `2026-08-03`
- 対象タブ `2026.8`

対象月のタブが存在しない場合は、自動作成せず停止します。
月タブの複製は数式・GAS・締め処理へ影響するため、別工程で設計します。

## 追加位置

1. 実施日・クライアント・店舗・実施時間の列を読取
2. クライアント列が空白の区切り行を除外
3. 最後の有効な案件行を雛形行に決定
4. その直後、区切り行やフッターの前へ新しい行を挿入

## 継承

雛形行から次をコピーします。

- 数式
- セル書式
- 入力規則
- 行の表示形式
- 条件付き書式の対象範囲

値そのものはコピーしません。

## 新しく書く値

- 実施日
- スタッフ名は空白
- クライアント
- 店舗
- メーカー
- メニュー
- 入店時間
- 実施時間
- 外注名
- スタッフ基本給
- 案件ID

請求・支払の他の入力値は、管理画面の安全編集から入力します。

## 公開

新しい行の追加と検算が完了するまで、案件は下書きです。

成功後に、作成時の指定へ戻します。

- 下書き
- 即時公開
- 公開予約
