# 新しい月タブの安全作成 v1.2

## 流れ

1. 作成先の月をYYYY.M形式で指定
2. 直前の月タブ、または指定した複製元を確認
3. 作成先が存在しないことを確認
4. 複製元の行数・列数・条件付き書式・保護範囲を取得
5. タブを複製
6. 入力列の値だけを初期化
7. 数式・入力規則・条件付き書式・保護範囲を照合
8. 合格した場合だけ完了
9. 不一致があれば作成したタブを削除

## 初期化するもの

列マッピングに登録された入力列を初期化します。

例:

- 実施日
- スタッフ名
- 体温・到着時刻
- クライアント・店舗・メーカー・メニュー
- 外注名
- 請求・支払の入力列
- 案件ID
- 提出状況等の入力列

## 残すもの

- 数式列
- セル書式
- 入力規則
- 条件付き書式
- 保護範囲
- 行数・列数
- 既存GAS

## 三重ロック

実際の作成には以下がすべて必要です。

1. `sheetMappings.shift.enabled=true`
2. `sheetMappings.shift.monthCreation.enabled=true`
3. `companyFeatureSettings.monthSheetCreationReady=true`

さらに `verifiedSpreadsheetId` が対象スプシIDと一致しない場合は作成できません。
