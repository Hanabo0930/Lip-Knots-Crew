# 安全なスプシ書込キュー v0.7

## 防御層
1. 会社ごとの`enabled`がtrueでなければ書かない
2. 操作ごとの許可項目以外は拒否
3. 案件ID列で行を検索
4. 案件IDがない場合、明示許可時だけ日付・クライアント・店舗・時間を再照合
5. 書込前の期待値と現在値が違えば競合停止
6. 対象セルだけ更新し、行全体は更新しない
7. 書込後に再読込して値を検証
8. 変更前後を監査ログへ保存
9. 一時障害だけ最大5回再試行
10. 競合・設定不足は自動再試行せず管理者確認

## 初期状態
`config-samples/sheet-mapping.safe-write.json`は`enabled:false`です。GAS安全監査と検証コピーでの金額突合が終わるまで有効化しません。
